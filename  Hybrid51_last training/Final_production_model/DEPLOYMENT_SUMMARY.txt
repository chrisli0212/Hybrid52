================================================================================
ARCHITECTURE SUMMARY
================================================================================

Stage 1: Per-Symbol Per-Agent Predictions
  - 35 independent models
  - Input: 325-dim features
  - Hidden: 64-dim
  - Output: Binary probability per symbol-agent pair

Stage 2: Cross-Symbol Fusion
  - 7 fusion models (one per agent)
  - Combines predictions across 5 symbols
  - Uses chain context (2D frozen features)
  - Output: Fused probability per agent

Stage 3: VIX-Gated Meta-Ensemble
  - 1 final model
  - Input: 7 agent probabilities + 10-dim VIX features
  - Learned per-agent gates conditioned on VIX regime
  - Output: Final directional probability

Total Parameters: ~23.8M (mostly in Stage 1)
Total Inference Time: ~2-3 seconds per split (on GPU)

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues:

1. "FileNotFoundError: checkpoint not found"
   → Verify all 45 files in models/ directory exist
   → Check paths in config/production_config.json

2. "CUDA out of memory"
   → Reduce batch_size in inference script
   → Use CPU mode: --device cpu

3. "Shape mismatch in Stage 2"
   → Ensure chain_context.npz exists in models/stage2/
   → Verify all Stage 1 models loaded successfully

4. "VIX features shape error"
   → Check VIX data has shape (N, 4, 10)
   → Script automatically collapses to (N, 10)

For detailed troubleshooting, see README.md sections:
  - "Monitoring and Diagnostics"
  - "Stage-by-Stage Validation"
